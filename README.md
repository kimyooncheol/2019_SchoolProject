# 사용된 기술
- PHP: Database와 연동 및 구글 차트로 json 데이터 보내기

- Arduino: -시계모드: 리얼타임 클록을 사용한 현제 시각및 온습도 센서 데이터 LCD출력

- GoogleChart: 측정된 심박수 그래프화
